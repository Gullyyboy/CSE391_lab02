# CSE391 Projects

This repository contains three small web projects for the CSE391 assignment:

- `fortune/` — Fortune Generator
- `stopwatch/` — Stopwatch
- `todolist/` — To-Do List

Each project has been organized with:

- `index.html`
- `style.css`
- `script.js`

## Notes

- The homepage is `index.html` at the repository root.
- Every project includes an **All Projects** link back to the homepage.
- The Fortune Generator preserves theme selection using `localStorage`.
- The To-Do List supports task persistence and Enter-key submission.
- The Stopwatch has a real-time decimal timer and button state handling.
